# 定位

- 是一种更加高级的布局手段。
- 通过定位，可以将元素播放到页面的任意位置。
- 使用 `position` 属性来设置定位，可选值：
  - `static`：默认值，元素是静止的，没有开启定位
  - `relative`：开启元素的相对定位
  - `absolute`：开启元素的绝对定位
  - `fixed`：开启元素的固定定位
  - `sticky`：开启元素的固定定位

## 偏移量 `offset`

当元素开启定位之后，可以通过偏移量来设置元素的位置。

> 只影响自身，不会影响其它元素的布局。(`margin`等会影响其它元素布局)

- `top`
- `bottom`
- `left`
- `right`

## 相对定位 `relative`

- 开启相对定位之后，如果不设置偏移量，元素不会发生任何变化
- 参照于元素在文档流中的位置进行定位的
- 相对定位会提升元素的层级
- 相对定位不会使元素脱离文档流
- 相对定位不会改变元素的性质——块还是块，行内还是行内
